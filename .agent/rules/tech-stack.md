## 2. Backend Stack

### Option 1: NestJS (Node.js)
*   **Framework:** NestJS (Latest)
*   **Language:** TypeScript
*   **ORM:** TypeORM or Prisma
*   **Validation:** class-validator, class-transformer
*   **Doc:** Swagger (OpenAPI)

### Option 2: Laravel (PHP)
*   **Framework:** Laravel 10/11
*   **Language:** PHP 8.2+
*   **ORM:** Eloquent
*   **Auth:** Laravel Sanctum
*   **Debug:** Laravel Telescope (Dev only)

### Option 3: Go (Golang)
*   **Framework:** Gin or Echo
*   **Language:** Go 1.21+
*   **ORM:** GORM or SQLC
*   **Config:** Viper
*   **Logger:** Zap or Zerolog

Only usage of libraries in this list is allowed. Discussions needed if adding new ones.
